## âš–ï¸ Legal & Ethical Considerations

### **Data Sources**
- All data must be **publicly available** or **FOIA-requestable**
- No hacking, no illegal access
- Cite sources transparently

### **Defamation Protection**
- Present **patterns**, not accusations
- Use "suspected" / "appears to" / "data suggests" language
- Allow for alternative explanations

### **Whistleblower Safety**
- Never expose sources without consent
- Recommend encrypted communication (Signal, ProtonMail)
- Link to legal protections (e.g., ZZPri in Slovenia)

### **Community Guidelines**
- No hate speech, no violence advocacy
- Focus on **systems**, not individuals (unless public officials)
- Encourage transparency, not vigilantism
